# Outputs

This folder is reserved for reproducible runtime outputs.

- root CSV/JSON/Markdown files: governed analytical tables, validation reports, and run manifests generated by the pipeline
- `analysis/`: optional organized analytical outputs after running `scripts/cleanup_repository.py`
- `profiling/`: optional organized profiling outputs after running `scripts/cleanup_repository.py`
- `visualizations/`: generated chart pack
- `warehouse/`: SQL warehouse logs and validation checks
- `release/`: release-readiness outputs
- `dashboard/`: generated local HTML dashboard and local vendor asset
